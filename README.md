## 1. Install
`npm install --save-dev babel-plugin-auto-argument`

## 2. Basic Usage
Edit your __.babelrc__ file:
```json
{
  "plugins": [
    "auto-argument"
  ]
}
```
In your js file:
```js
[1, 2, 3].map(_ * 2)
```
And it will be compiled to:
```js
[1, 2, 3].map(function (_) { 
    return _ * 2
})
```

In your js file:
```js
const a = [
    [1, 2],
    [3, 4, 5],
    [6, 7, 8, 9]
].filter(_.length <= 3).map(_.filter(10 % _ === 0));
console.log(a);
```
And it will print:
```js
[
    [1, 2],
    [5]
]
```

## 3. Usage with custom variable name
Edit your __.babelrc__ file:
```json
{
  "plugins": [
    "auto-argument",
    {
      "variableName": "$"
    }
  ]
}
```
In your js file:
```js
[1, 2, 3].map($ * 2)
```
